Git 上传 Github 教程（部分）

2024-05-25

Git 上传 Github 教程（部分）

（其他问题不讨论：如何科学上网？如何安装Git？……）

  1. 本地新建文件夹

  2. cmd，执行 git init，将文件夹变成git仓库

  3. 将文件移动到此文件夹，执行 git add .，将文件添加到git仓库

  4. SSH_key密钥……懒得说啦。百度一下吧：）

  5. 执行 git commit -m "xxx"，确认添加，其中-m是添加commit的描述，可以不加，不加的话，默认是空的。

  6. 关联本地仓库与github：执行 git remote add origin https://github.com/xxx/xxx.git（此链接为github的链接，使用ssh或https都可以）

  7. 执行 git push -u origin main 由于新建的远程仓库是空的，所以要加上-u这个参数，等远程仓库里面有了内容之后，下次再从本地库上传内容的时候只需下面这样就可以了：git push origin main
